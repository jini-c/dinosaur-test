⚙️ 핵심 기술 구현 - JavaScript

🔄 1. 페이지 전환 시스템

function showPage(pageId) {
  // 모든 페이지 숨기기
  document.querySelectorAll('.page')
    .forEach(page => page.classList.add('hidden'));
  // 선택된 페이지만 보이기
  document.getElementById(pageId + '-page')
    .classList.remove('hidden');
}

🔧 개발 과정의 주요 도전과 해결

❌문제 1: 페이지 전환이 안 되는 오류

문제: showPage() 호출했는데 화면이 바뀌지 않음

원인: DOM 업데이트 타이밍과 함수 호출 순서

문제: 공룡 이미지가 표시되지 않음
